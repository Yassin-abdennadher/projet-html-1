En Résumé

  • HTML (HyperText Markup Language) est le langage utilisé pour structurer le contenu d’une page web.
  • Il définit la structure (titres, paragraphes, images, liens, listes, formulaires, etc.).
  • Il n’est pas un langage de programmation, mais un langage de balisage (avec des balises ouvrantes et fermantes).
  • Pour la mise en forme, on utilise généralement CSS à côté.
